Let’s make the reference group for the regressions the major with the lowest GPA on average.

# Build df
SLR_GPA_Major <- HSLS_Data |>
  filter(X5GPAALL >= 0,
         X4ENTRYMAJ23 != "Unit non-response") |>
  filter(X4ENTRYMAJ23 != "Item legitimate skip/NA") |>
  filter(X4ENTRYMAJ23 != "Missing") |>
  select(X5GPAALL, X4ENTRYMAJ23)

# set reference group
SLR_GPA_Major$X4ENTRYMAJ23 <- relevel(SLR_GPA_Major$X4ENTRYMAJ23, ref = "Undeclared/undecided")

Conveniently, that is undeclared/undecided, which is a good comparison.

Let’s look at relationship between high school GPA and college GPA organized by prospective major.

I like this version of the regression. The results imply that career choice when in 9th grade has little implication for college GPA, except in a few select fields. The only positive boost to college GPA is for kids wanting a job in education. I think it makes sense that kids with higher grades would want to be involved in education in the future because they are already successful in that environment.

I think school engagement might be correlated with college GPA, particularly among kids who want a career in education.
